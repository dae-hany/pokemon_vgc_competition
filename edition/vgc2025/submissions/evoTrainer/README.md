# EvoTrainer

This project contains the EvoTrainer, an agent designed to compete in
the [VGC AI Competition 2025](https://gitlab.com/DracoStriker/pokemon-vgc-engine/-/wikis/VGC-AI-Competition/2025). It
can easily be used via the `EvoCompetitor` class which is located in the `evo_agent` folder.

An example usage can be found in the `evo_agent/test_match` file, where two EvoTrainers compete against each other in a
match.

Have fun experimenting!
