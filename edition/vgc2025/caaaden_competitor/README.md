Caaaden Pokemon VGC Competitor
==============================

The folder contains two code files:

* `caaaden_competitor.py`: Contains the BattlePolicy and Competitor Class.
* `caaaden_main.py`: A script that runs the remote Competitor.

How to Run the Caaaden Bot
--------------------------

* Navigate to the directory where the `caaaden_main.py` file is located in the terminal.
* Run the following command: `python caaaden_main.py --id <agent_id>`
* Replace with the ID that matches your agent.
